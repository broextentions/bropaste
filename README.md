## üöÄ Overview

**BroPaste** helps you save time, reduce typing errors, and stay productive. Save frequently used text and quickly insert it into any input field using the context menu or popup. 

Perfect for registration forms, login pages, surveys, feedback forms, customer support replies, and everyday repetitive typing.

## ‚ú® Features

- **üñ±Ô∏è Context Menu Integration** Right-click any input field and insert saved text instantly without opening the extension.

- **üß† Smart Paste Engine** Works flawlessly even on complex websites (like Google Forms, React apps) where standard pasting often fails.

- **üìù Unlimited Entries** Store emails, usernames, addresses, phone numbers, code templates, and more.

- **üé® Theme Support** Automatically adapts to **Light** or **Dark** mode based on your browser preferences.

- **üîÑ Drag & Drop Sorting** Organize your snippets easily to keep important ones at the top.

- **üìã One-click Copy** Quickly copy values to your clipboard directly from the popup.

- **üíæ Import & Export** Backup your data or transfer it between browsers and devices (JSON format).

- **üîí 100% Local Storage** Your data stays on your device. No cloud, no tracking.

## üõ† How to Use

1. Click the extension icon to add new text snippets.
2. **Right-click** on any input field on a website.
3. Hover over the **BroPaste** menu item and select your text.
4. Done!

## üì¶ Installation

1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked**.
5. Select the folder containing the `manifest.json` file.
